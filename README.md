# Stirix

Stirix is a football-focused web platform for articles, comments, quizzes, rankings, and interactive football tools.

The project is built as a full-stack application with a TypeScript-based Next.js frontend and an Express/MongoDB backend. It includes public article reading, authenticated user features, writer/admin workflows, Google authentication, comment safety rules, and football-focused interactive sections.

## Features

- Football article feed with categories
- Full article pages with metadata, views, hearts, and related articles
- User accounts and Google authentication
- Writer/admin article approval workflow
- Article comments with moderation-oriented safety rules
- Football quiz with rankings
- Team tools and saved team lists
- Multilingual Romanian/English interface
- SEO support with dynamic metadata and structured data
